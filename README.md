# 🚀 Emoji Prediction from Text  
### Comparative Analysis of LR, LSTM, and BERTweet  

---

## 📌 Overview  
This project predicts the most relevant emoji for a given tweet using NLP techniques.

Emojis carry strong sentiment and contextual meaning, and automating their prediction helps in:
- Sentiment Analysis  
- Sarcasm Detection  
- Conversational AI  

This project compares:
- Logistic Regression (Baseline)
- LSTM (Sequential Model)
- BERTweet (Transformer Model)

---

## 🎯 Problem Statement  
Given a tweet, predict the most appropriate emoji from 20 classes.

### Challenges:
- Informal text & slang  
- Short context (tweets)  
- Class imbalance  
- Sarcasm and ambiguity  

---

## 📊 Dataset  
- **Dataset**: SemEval-2018 Task 2  
- **Train**: 42,500 samples  
- **Validation**: 7,500 samples  
- **Test**: 50,000 samples  
- **Classes**: 20 emojis
